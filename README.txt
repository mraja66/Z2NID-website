Struktur:
- index.html        : Halaman Produk (dengan tombol Beli/Pesan pada foto)
- transaksi.html    : Halaman input transaksi (form)
- invoice.html      : Halaman struk/invoice (siap cetak)
- kontak.html       : Halaman "Kontak Kami" & Biodata
- tentang.html      : Halaman Tentang (opsional)
- styles.css        : Style global, responsive
- script.js         : Data produk, logika transaksi & invoice
- assets/           : Gambar logo, banner, placeholder produk & foto biodata
- favicon.ico       : Favicon

Kebutuhan Tugas yang Terpenuhi:
- Tema penjualan akun MLBB (barang/jasa).
- Bagian Header, Menu, Sidebar (left), Content, Footer.
- >= 4 halaman (Produk, Transaksi, Invoice, Kontak; plus Tentang).
- Tombol Beli di foto (overlay bottom center) -> ke halaman transaksi.
- Form Transaksi berisi input teks, email, telp, select, radio, checkbox, textarea.
- Setelah Checkout -> tampil Invoice dengan desain rapi. Bisa cetak/print.
- Ada <marquee> dan link gambar (banner di sidebar mengarah ke WhatsApp).
- Desain responsive (mobile-first) untuk Android/iPhone/Windows/Mac.
- Tanpa library eksternal; murni HTML, CSS, JS.
